### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/haliljon/hello-rails-back-end.git

  cd hello-rails-back-end

  bundle install

  rails db:create db:migrate db:seed

 ./bin/dev
```
